# 剧本分析报告

## 整体风格建议
**写实纪实风**（Realistic Documentary Style），参考《我在故宫修文物》《国家宝藏》质感。
- 暖色调，侧光/逆光勾边，强调手部与器物的细节
- 浅景深特写为主，慢节奏，匠人动作写实
- 字幕字体建议用思源宋体/楷体类传统中式字体，贴合非遗主题
- 8:1 镜头语音比，旁白语速平稳偏慢，给画面留呼吸感

## 人物清单

### 1. 匠人（掐丝珐琅技艺传承人）
- **身份**：传统手工艺人，掐丝珐琅技艺传承人
- **外貌描述**：中年男性，约 50 岁出头，面容沉稳、目光专注；短发花白，眉毛浓密；手部是重要特征——指节粗大、带着常年劳作留下的细茧与些许铜锈痕迹
- **服装风格**：深灰色/藏青色棉麻中式立领对襟上衣，做旧质感；袖口可挽起露出小臂
- **性格特点**：专注、沉稳、内敛；动作缓慢而精确
- **AI 生图提示词**：
```

```

## 场景清单

### 1. 工坊内景（主场景，01/03/04/05/07 共用）
- **场景类型**：室内
- **环境描述**：传统中式手工作坊，木梁结构，自然光从高窗斜射入内；墙边挂着一排待完成的铜胎半成品；案台上有锤、镊子、釉料碟、砂石等工具；地面是青砖，整体陈旧但整洁
- **关键元素**：铜胎、釉料碟、镊子、铜丝卷、案台、工具墙
- **色调氛围**：暖琥珀色 + 铜色，温暖宁静，岁月感
- **AI 生图提示词**：
```
Interior of a traditional Chinese cloisonné enamel workshop, wooden beams, high windows with warm natural sunlight streaming in, copper blanks hanging on the wall, workbench with tools (hammers, tweezers, enamel dishes, sandstones), blue-gray brick floor, warm amber and copper color palette, peaceful and timeless atmosphere, realistic documentary photography style, wide shot, cinematic lighting, 8K
```

### 2. 案头画稿（02 用）
- **场景类型**：室内特写

- **关键元素**：宣纸、毛笔、墨锭、镇纸、砚台
- **色调氛围**：水墨淡雅，柔和漫射光
- **AI 生图提示词**：
```

```

### 3. 窑炉工位（06 用）
- **场景类型**：室内
- **环境描述**：砖砌窑炉的特写，炉口微微透着橘红色火光；周围散落着几片刚出窑尚有余温的铜胎
- **关键元素**：窑炉炉口、火焰/余温感、铜胎半成品
- **色调氛围**：暗调 + 暖橘红高光，强明暗对比，温度感
- **AI 生图提示词**：

